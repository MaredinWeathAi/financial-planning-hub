   - BalanceSheetData: Assets, liabilities, equity with ratios
   - CashFlowData: Operating, investing, financing cash flows
   - BusinessFinancials: Container for all three statements
   
   Accounts & Debt:
   - BusinessAccount: Bank and investment accounts
   - BusinessDebt: Loans and lines of credit
   
   Planning:
   - BusinessGoal: Business objectives and targets
   - BusinessPlan: Complete integrated business profile
   
   Features:
   ✓ Automatic ID generation with SHA256 hashing
   ✓ Full serialization/deserialization (to_dict/from_dict)
   ✓ Enum support for common values (EntityType, DebtType, GoalPriority)
   ✓ Calculated properties and metrics
   ✓ Aggregated metrics calculation

   File: biz_pdf_parser.py (823 lines, 33KB)
   ─────────────────────────────────────────
   Intelligent PDF extraction engine with auto-detection:
   
   Main Classes:
   - BusinessPDFImporter: Core parser with extraction methods
   - BusinessDataStore: JSON-based persistence layer
   
   Key Methods:
   - parse_pdf(filepath): Main entry point with auto-detection
   - extract_business_profile(): Company name, EIN, address, industry
   - extract_income_statement(): Revenue, COGS, expenses, net income
   - extract_balance_sheet(): Assets, liabilities, equity, ratios
   - extract_cash_flow(): Operating, investing, financing activities
   - extract_tax_data(): Tax return specific fields (Forms 1120, 1120S, 1065)
   - extract_accounts(): Bank and investment accounts with balances
   - extract_debt_schedule(): Loans and credit lines with terms
   
   Helper Functions:
   - clean_number(): Converts various number formats
   - extract_date(): Recognizes common date patterns
   - extract_ein(): Extracts EIN (XX-XXXXXXX format)
   - detect_document_type(): Auto-detects PDF type
   - parse_business_pdf(): Quick wrapper function
   - batch_import_pdfs(): Process multiple PDFs from directory
   
   Features:
   ✓ Auto-detects document type (tax return, P&L, balance sheet, cash flow, etc.)
   ✓ Handles multiple number formats ($1.5M, $500K, (1234), etc.)
   ✓ Recognizes various date formats (MM/DD/YYYY, YYYY-MM-DD, etc.)
